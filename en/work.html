Z.UST

Zapret.rUST (or Z.UST) stands for "Zapret GUI written in Rust" - hence the name. Ultimately, the project evolved into a universal solution for bypassing censorship in both Russia and Belarus, featuring a built-in free proxy, deep zapret configuration, and an intuitive interface. The project is still a W.I.P, with mobile development planned for the future.
